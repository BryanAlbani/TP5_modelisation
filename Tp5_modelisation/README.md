**L'exercice 1 Chaikin :** est directement visible dans la scène comme demandé dans l'exercice. 

Il est possible d'ajouter manuellement des points de contrôles, il faut créer un nouvel objet et l'ajouter à la liste de points de contrôles du script.

En fonction du nombre d'itération, l'interpolation de la courbe sera plus ou moins smooth.



**L'exercice 2 subdivision loop :** Pour que le script soit actif, il faut faire play. 

Dans la scène j'ai mis 2 objets un simple (cube) et un plus complexe (balle à pique), chacun est dédoublé pour voir l'action d'une itération ou de 6. 

Attention si vous souhaitez, faire le test sur un autre objet, il faut que là où le script est placé, il est le mesh filter.
